3. Features & Requirements

3.1 Core Features

WhatsApp/USSD Registration
  • Collect: Name, skill, location, M-Pesa number
  • AI: Auto-categorize skills from voice notes (NLP)
Profile Verification
  • ID scan (Jumio API)
  • Skills quiz (WhatsApp chatbot)
Post Jobs via App/USSD
  • Fields: Job type, budget, location, urgency
AI Suggestions
  • Auto-complete job descriptions
Inputs
  • Job requirements
  • Worker skills
  • Location
  • Ratings
Output
  • Ranked list of top 5 workers
Algorithm
  • Collaborative filtering + logistic regression

3.2 Advanced AI Features (Phase 2)

Predictive Demand Alerts

SMS workers about service hotspots based on historical demand patterns.

Voice-Based Skills Verification

AI checks technical knowledge through voice responses to questions.

Fraud Detection

Flags fake profiles and potential scams using behavioral analysis.
